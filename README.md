# Lineare Algebra und Geometrie (LAG)

This repository contains learning materials, notebooks, and exercises for the **Linear Algebra and Geometry** course.  
The focus is on **linear equation systems, matrices, Gaussian elimination, and Python tools (NumPy / SymPy)**.
